# Degenerate-Blackjack-
Degenerate BlackJack is a simple, text-based Blackjack game written in Python. The game lets players experience the thrill of hitting or standing against a dealer, complete with ASCII-style card visuals. Multiple decks are shuffled together for a more realistic casino experience, and players can decide to hit or stand to play it safe. This game has some features 
including: 
* Customizable player name
* Six-deck shoe for realistic gameplay
* ASCII card display for a visual touch
* Player decision loop for multiple hits
* Dealer’s face-down card revealed after player stands

How to play: 
1. Run python blackjack.py
2. Follow the prompts to hit or stand
3. Try to get as close to 21 as possible without going over!
